# SIDStation
The SIDStation project is an open-source scientific platform intended for continuous monitoring of Very Low Frequency (VLF) transmitters
in order to detect Sudden Ionospheric Disturbances (SID).
The system is designed around a Raspberry Pi 4, an SDRplay RSP2 receiver and an active magnetic loop antenna.
Measurements are processed in real time and stored in InfluxDB, while Grafana provides visualization of historical and live data.
